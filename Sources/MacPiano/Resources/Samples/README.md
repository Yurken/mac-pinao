This project supports fixed piano samples placed in:

`Sources/MacPiano/Resources/Samples/acoustic_grand_piano/*.mp3`

Download helper:

```bash
bash scripts/download_piano_samples.sh
```

Sample source:

- Repository: `gleitz/midi-js-soundfonts`
- Instrument: `FluidR3_GM/acoustic_grand_piano-mp3`
- URL: https://github.com/gleitz/midi-js-soundfonts

Notes:

- Files are not checked in by default.
- If samples are missing, the app automatically falls back to synthesized audio.
